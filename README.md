# FutureEngine
FutureEngine is the 2D engine being used to make Future Calling. It is open-source and free to use.

The engine is a static library, so to use it you create a empty C++ console project, then link to the engine.

Over time I will add documentation/instructions on how to create and set up your own project and games using it.

*Note* - still *very* early in development, but I will make commits daily.
